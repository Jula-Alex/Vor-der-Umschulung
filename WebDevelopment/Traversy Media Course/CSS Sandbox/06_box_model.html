Box One

Se apri il development tool vedrai che ci sono margin e padding di default per questo c'e lo spazio intorno agli elementi, questo si puo ovviamente modificare con css, puoi usare i selettori e poi impostare a 0, in questo modo il default rimarra ma le tue impostazioni messe in css sovrascriveranno quelle di default

Box Two

Per resettare tutti gl'elementi a 0 basta mettere un *{} e impostare margin e padding a 0, la regola e' sempre la stessa di quella sopra, le nuove impostazioni verranno sovrascritte a quelle di default

Box Three

Se si aggiunge un padding grande e la largezza width del div e piccola allora il padding si aggiungera alla largezza del div creando problemi, per soluzionare questo problema bisogna impostare al * box-sizing = border-box; In questo modo il padding invece di spingere i bordi verso l'esterno violando le misure impostate lo fara verso l'interno mantenendo le misure impostate.
