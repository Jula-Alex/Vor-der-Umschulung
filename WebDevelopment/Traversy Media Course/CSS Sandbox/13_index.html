La differenza tra display none e visibility none e che il primo toglie proprio l elemento mentre il secondo lo fa diventare solo invisibile lasciando uno spazio vuoto

Normalmente in css le proprieta vengono eseguite in ordine dall alto verso il basso, ma se lo si fa targettando una classe .hello ed un elemento h1 per esempio anche se in css h1 sara al di sotto di .hello, verra comunque eseguito .hello e questo perche le classi sovrascrivono

Se invece si vuole per forza eseguire quel specifico elemento basta aggiungere "!important" in css

Si possono usare anche valori negativi come nel esempio sopra di hello world ma e consigliato di non farlo, qui lo facciamo solo per mostrare che lo si puo fare solo con margin pero, con padding non funzionera
